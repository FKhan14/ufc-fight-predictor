# UFC Fight Outcome Predictor with ROI Analysis

**Goal:** Build a machine learning system to predict UFC fight outcomes and analyze betting profitability using custom ML implementations from scratch.

## Project Status
**In Progress** - Currently in initial setup phase

## Project Structure
```
ufc-fight-predictor/
├── data/
│   ├── raw/              # Original UFC dataset
│   └── processed/        # Cleaned and feature-engineered data
├── notebooks/            # Jupyter notebooks for analysis
├── src/                  # Source code
│   ├── ml_from_scratch/  # Custom ML implementations
│   ├── features/         # Feature engineering functions
│   └── evaluation/       # Model evaluation utilities
├── models/               # Saved trained models
└── results/              # Visualizations and reports
```

## Dataset
- **Source:** [Ultimate UFC Dataset on Kaggle](https://www.kaggle.com/datasets/mdabbert/ultimate-ufc-dataset)
- **Size:** ~6,000 fights (2010-2024)
- **Features:** Fighter statistics, betting odds, fight outcomes

## Machine Learning Approaches
All core algorithms implemented from scratch:
- Logistic Regression
- Random Forest
- Neural Networks
- Ensemble Methods
